## 最終設計方案：夢幻夜空風格 6宮格圖文選單

### 設計理念
- **品牌定位**：優雅、夢幻、高級感，符合醫美產業「讓美麗自然發生」的理念
- **色彩心理學**：深藍色（信任、專業）+ 紫色（優雅、高貴）+ 金色（尊貴、夢幻）
- **視覺一致性**：與官網 https://yochillsaas.com/ 的燙金深藍底尊爵感保持一致

### 6宮格布局配置

```
┌─────────────┬─────────────┬─────────────┐
│  📅 立即預約  │  🔍 查詢預約  │  👤 會員中心  │
│  (流星區)    │  (星星區)    │  (月亮區)    │
├─────────────┼─────────────┼─────────────┤
│  🎁 優惠活動  │  🎮 小遊戲   │  💬 客服諮詢  │
│  (星星區)    │  (流星區)    │  (月亮區)    │
└─────────────┴─────────────┴─────────────┘
```

### 功能說明

| 位置 | 功能 | 動作類型 | 跳轉目標 |
|------|------|----------|----------|
| 左上 | 📅 立即預約 | uri | LIFF 預約頁面 |
| 中上 | 🔍 查詢預約 | uri | LIFF 預約查詢頁面 |
| 右上 | 👤 會員中心 | uri | LIFF 會員中心頁面 |
| 左下 | 🎁 優惠活動 | uri | LIFF 優惠活動頁面 |
| 中下 | 🎮 小遊戲 | uri | LIFF 小遊戲頁面（一番賞、拉霸、轉珠） |
| 右下 | 💬 客服諮詢 | message | 觸發 AI 客服機器人 |

- **顏色**：白色（#FFFFFF）+ 金色描邊（#FFD700）

- **圖示大小**：32px × 32px
- **圖示顏色**：白色（#FFFFFF）+ 金色光暈（#FFD700）
