# YOChiLL 醫美診所 SaaS 平台 - 官網前端分析報告

## 官網資訊
- **網址**: https://yochillsaas.com
- **標題**: YOChiLL 醫美診所 SaaS 平台
- **主標語**: 繁瑣的流程交給數據，有溫度的服務留給顧客

## 前端架構分析

### 主要功能模組
1. **功能特色**: 展示系統核心功能
2. **關於我們**: 介紹公司與團隊
3. **方案價格**: 展示定價方案
4. **登入**: 使用者登入入口
5. **免費試用**: 14 天免費試用註冊

### 核心 CTA (Call-to-Action)
1. **立即預約，開啟您的專屬美麗旅程**: 主要預約入口
2. **立即免費試用 14 天**: 免費試用註冊
3. **登入**: 現有使用者登入

### 與後台對接點

#### 1. 登入功能
- **前端**: 登入按鈕 → 導向登入頁面
- **後台**: `/api/auth/official-login` (SSO 單一登入)
- **流程**: LINE Login → JWT Token → 跨域 Cookie → 後台驗證

#### 2. 免費試用註冊
- **前端**: 免費試用按鈕 → 導向註冊頁面
- **後台**: `/api/auth/register` (使用者註冊)
- **流程**: 填寫診所資訊 → 建立帳號 → 發送歡迎信 → 啟用 14 天試用

#### 3. 立即預約
- **前端**: 立即預約按鈕 → 導向預約頁面
- **後台**: `/api/appointments/create` (預約建立)
- **流程**: 選擇診所 → 選擇療程 → 選擇時間 → 填寫資料 → 確認預約

#### 4. LINE LIFF 整合
- **LIFF ID**: 2008492658-YVUxvCIJ
- **LIFF URL**: https://liff.line.me/2008492658-YVUxvCIJ
- **流程**: LINE 中開啟 LIFF → 登入驗證 → 完整預約流程
