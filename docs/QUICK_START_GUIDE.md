# YOChiLL 快速入門指南

> **5 分鐘上手核心功能**

---

## 🚀 第一步：登入系統

1. 開啟系統網址
2. 點擊「登入系統」按鈕
3. 使用 Manus OAuth 完成登入
4. 系統自動導向診所儀表板

---

## 📊 儀表板總覽

登入後首先看到的是**診所儀表板**（`/clinic`），這裡顯示：

| 區塊 | 說明 |
|------|------|
| 今日預約 | 當天的預約數量與狀態 |
| 本月營收 | 當月累計營收與目標達成率 |
| 待處理事項 | 需要關注的待辦事項 |
| 快速操作 | 常用功能的快捷入口 |

---

## 👥 客戶管理（最常用）

**路徑**：側邊欄 → 客戶管理

### 新增客戶
1. 點擊右上角「+ 新增客戶」
2. 填寫姓名、電話（必填）
3. 選擇會員等級
4. 點擊「儲存」

### 搜尋客戶
- 在搜尋框輸入姓名或電話
- 使用篩選器依會員等級過濾

### 查看客戶詳情
- 點擊客戶姓名進入詳情頁
- 可查看：消費記錄、預約歷史、療程記錄

---

## 📅 預約管理

**路徑**：側邊欄 → 預約管理

### 新增預約
1. 點擊「+ 新增預約」
2. 選擇客戶（可搜尋）
3. 選擇服務項目
4. 選擇醫師/美容師
5. 選擇日期與時段
6. 點擊「確認預約」

### 預約狀態管理
| 狀態 | 操作 |
|------|------|
| 待確認 | 點擊「確認」 |
| 已確認 | 客戶到診後點擊「完成」 |
| 爽約 | 客戶未到點擊「爽約」 |

---

## 💉 療程記錄

**路徑**：側邊欄 → 療程記錄

### 新增療程記錄
1. 點擊「+ 新增記錄」
2. 選擇客戶與預約
3. 填寫療程詳情
4. 上傳前後對比照片（可選）
5. 設定滿意度評分
6. 點擊「儲存」

---

## 📦 產品管理

**路徑**：側邊欄 → 產品管理

### 產品類型
- **服務**：療程項目（無庫存）
- **產品**：實體商品（有庫存）
- **套餐**：療程組合包

### 新增產品
1. 點擊「+ 新增產品」
2. 選擇類型
3. 填寫名稱、價格
4. 設定庫存數量（產品類型）
5. 點擊「儲存」

---

## 👨‍💼 員工管理

**路徑**：側邊欄 → 員工管理

### 新增員工
1. 點擊「+ 新增員工」
2. 填寫基本資料
3. 選擇職位
4. 設定薪資結構
5. 點擊「儲存」

---

## 📈 報表分析

**路徑**：側邊欄 → 報表分析

### 可查看的報表
- **營收報表**：日/週/月營收統計
- **預約統計**：預約數量與到診率
- **客戶分析**：新客/回客比例
- **員工業績**：業績排行榜

---

## 🔔 通知設定

**路徑**：側邊欄 → 通知管理

### 自動通知類型
- 預約提醒（前 1 天）
- 療程後關懷
- 生日祝福
- 低庫存警示

---

## 💡 小技巧

1. **快速搜尋**：在任何列表頁面都可使用搜尋框
2. **批量操作**：勾選多筆資料後可批量處理
3. **匯出資料**：點擊「匯出」按鈕下載 Excel
4. **側邊欄收合**：點擊左上角按鈕收合側邊欄
