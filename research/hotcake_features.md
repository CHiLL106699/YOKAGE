# 夯客 Hotcake 核心功能研究

## 產品定位
- 美業專用的預約、會員、POS 系統
- 目標客群：美髮、美甲、美睫、美容、按摩整復、醫美診所
- 每月 0 元起，月繳不綁約

## 核心功能模組

### 1. 會員系統
- 即時掌握顧客動態
- 用數據打造會員忠誠計畫
- 會員護照（上傳客人每次成品）
- 會員等級管理

### 2. 預約系統
- 整合線上下預約
- 自動化預約（自動計算服務時間和可預約時間）
- 即時且準確，不會有預約衝突
- 註冊到預約 60 秒完成

### 3. 業績管理
- 精準記錄每一筆消費
- 清楚做好業績統計
- 不再為結算而苦惱

### 4. 行銷再應用
- 票券管理
- 儲值金
- 紅利點數
- 會籍管理

## LINE 整合
- LINE 官方認同的合作夥伴
- 建立圖文選單
- LINE 通知和提醒
- 會員分眾行銷

## 適用產業
- 美髮沙龍
- 寵物美容
- 美容美顏
- 醫美診所
- 美甲美睫
- 運動瑜伽
- SPA 按摩

## 成效數據
- 續約率：99%
- 合作商家：2000+
- 總預約數：2.5M+
- 接單率上升 15%
- 營收上升 20%
- 行政效率提升 60%
- 爽約率下降 30%
- 會員忠誠度提升 40%

## 客戶痛點解決
- 數位化猶豫不決 → 直覺操作不費力
- 繁雜行政作業 → 效率提升 60%
- 沒有定期整理顧客資料 → 即時掌握客人動態
