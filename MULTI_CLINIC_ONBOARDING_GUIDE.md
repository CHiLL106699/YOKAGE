## 導入概覽

本指南旨在協助新診所快速導入 YOChiLL 醫美診所 SaaS 平台。整個導入流程分為 **5 個階段**，預計需要 **2-4 週**完成。本平台採用多租戶架構，支援多診所獨立資料隔離，確保資料安全與隱私。

**導入流程總覽**：
1. **前期準備**（1-3 天）：收集診所資訊、準備資料、設定帳號
2. **系統設定**（2-5 天）：設定診所資訊、組織架構、權限管理
3. **資料匯入**（3-7 天）：匯入客戶資料、療程資料、員工資料
4. **整合測試**（3-5 天）：測試 LINE Bot、金流支付、API 整合
5. **上線培訓**（2-3 天）：員工培訓、試運行、正式上線

### 1.4 LINE 官方帳號準備

若需使用 LINE Bot 功能，請準備以下資訊：

**LINE Developers 設定**：

1. **建立 LINE Official Account**
   - 前往 [LINE Official Account Manager](https://manager.line.biz/)
   - 建立新的官方帳號
   - 記錄 Basic ID（例如：@939tfrla）

2. **建立 Messaging API Channel**
   - 前往 [LINE Developers Console](https://developers.line.biz/console/)
   - 建立新的 Provider（若尚未建立）
   - 建立新的 Messaging API Channel
   - 記錄以下資訊：
     - Channel ID
     - Channel Secret
     - Channel Access Token（Long-lived）

3. **設定 Webhook URL**
   - 在 Messaging API 設定頁面
   - 設定 Webhook URL：`https://your-domain.com/api/line/webhook`
   - 啟用 "Use webhook"
   - 停用 "Auto-reply messages"（由系統控制回覆）

4. **建立 LIFF 應用**（選填，若需使用 LIFF 預約功能）
   - 在 LINE Developers Console 建立 LIFF App
   - 設定 Endpoint URL：`https://your-domain.com/liff`
   - 記錄 LIFF ID

## 階段二：系統設定（2-5 天）

### 2.1 建立診所帳號

**由系統管理員執行**：

1. **登入管理後台**
   - 使用超級管理員帳號登入
   - 前往「診所管理」→「新增診所」

2. **填寫診所基本資訊**
   ```
   診所名稱：[診所名稱]
   統一編號：[統一編號]
   負責人姓名：[負責人姓名]
   電話：[電話]
   地址：[地址]
   Email：[Email]
   ```

3. **上傳品牌資訊**
   - 上傳 Logo 圖檔
   - 設定品牌色彩（選填）
   - 設定診所簡介（選填）

4. **設定營業資訊**
   ```
   營業時間：
   - 週一至週五：09:00 - 18:00
   - 週六：09:00 - 12:00
   - 週日：休診
   
   休診日：
   - 國定假日
   - 特殊休診日（可自訂）
   ```

5. **建立診所管理員帳號**
   ```
   姓名：[管理員姓名]
   Email：[管理員 Email]（作為登入帳號）
   角色：診所管理員
   ```

### 2.2 設定組織架構

**由診所管理員執行**：

1. **建立部門**（選填）
   - 前往「組織架構」→「新增部門」
   - 範例：醫療部、行政部、行銷部

2. **建立角色**（若標準角色不符需求）
   - 前往「角色權限管理」→「新增角色」
   - 設定角色名稱與權限範圍

3. **建立員工帳號**
   - 前往「員工管理」→「新增員工」
   - 填寫員工基本資訊
   - 指派角色與部門
   - 系統會自動發送邀請 Email

### 2.3 設定療程與服務項目

**由診所管理員執行**：

1. **建立療程分類**
   - 前往「療程管理」→「療程分類」→「新增分類」
   - 範例：注射類、雷射類、保養類

2. **建立療程項目**
   - 前往「療程管理」→「新增療程」
   - 填寫療程資訊：
     ```
     療程名稱：[療程名稱]
     療程分類：[選擇分類]
     療程價格：[價格]
     療程時長：[時長（分鐘）]
     療程說明：[說明]
     注意事項：[注意事項]
     ```

3. **建立療程套餐**（選填）
   - 前往「療程管理」→「療程套餐」→「新增套餐」
   - 選擇包含的療程項目
   - 設定套餐價格與優惠

### 2.4 設定 LINE Bot

**由系統管理員執行**：

1. **設定 LINE Channel 資訊**
   - 前往「系統設定」→「LINE 整合」
   - 填寫 LINE Channel 資訊：
     ```
     Channel ID：[Channel ID]
     Channel Secret：[Channel Secret]
     Channel Access Token：[Channel Access Token]
     Basic ID：[Basic ID]
     ```

2. **設定 Webhook URL**
   - 系統會自動顯示 Webhook URL
   - 複製 URL 並在 LINE Developers Console 設定

3. **測試 LINE Bot 連線**
   - 點擊「測試連線」按鈕
   - 確認連線成功

4. **設定 LIFF 應用**（選填）
   - 填寫 LIFF ID
   - 測試 LIFF 應用開啟

### 2.5 設定金流支付

**由系統管理員執行**：

1. **選擇金流服務商**
   - 前往「系統設定」→「金流支付」
   - 選擇要啟用的金流服務商

2. **填寫金流金鑰**
   - 填寫 API Key、Merchant ID 等資訊
   - 選擇測試模式或正式模式

3. **測試金流連線**
   - 點擊「測試連線」按鈕
   - 確認連線成功

4. **設定付款方式**
   - 啟用信用卡、ATM、超商付款等方式
   - 設定手續費負擔方式（商家負擔或客戶負擔）

## 階段三：資料匯入（3-7 天）

### 3.1 匯入客戶資料

**由診所管理員執行**：

1. **下載客戶資料範本**
   - 前往「客戶管理」→「匯入客戶」
   - 下載 Excel 範本

2. **填寫客戶資料**
   - 依照範本格式填寫客戶資料
   - 確認必填欄位已填寫
   - 確認資料格式正確

3. **上傳客戶資料**
   - 上傳填寫好的 Excel 檔案
   - 系統會自動驗證資料格式
   - 確認匯入結果

4. **處理匯入錯誤**（若有）
   - 下載錯誤報告
   - 修正錯誤資料
   - 重新上傳

**注意事項**：
- 建議分批匯入（每批 500-1000 筆）
- 匯入前請備份原始資料
- 匯入後請檢查資料完整性

### 3.2 匯入療程記錄

**由診所管理員執行**：

1. **下載療程記錄範本**
   - 前往「療程管理」→「匯入療程記錄」
   - 下載 Excel 範本

2. **填寫療程記錄**
   - 依照範本格式填寫療程記錄
   - 確認客戶姓名或客戶編號正確
   - 確認療程名稱與系統中的療程一致

3. **上傳療程記錄**
   - 上傳填寫好的 Excel 檔案
   - 系統會自動關聯客戶與療程
   - 確認匯入結果

### 3.3 匯入員工資料

**由診所管理員執行**：

1. **下載員工資料範本**
   - 前往「員工管理」→「匯入員工」
   - 下載 Excel 範本

2. **填寫員工資料**
   - 依照範本格式填寫員工資料
   - 確認 Email 格式正確（作為登入帳號）
   - 確認角色名稱與系統中的角色一致

3. **上傳員工資料**
   - 上傳填寫好的 Excel 檔案
   - 系統會自動建立員工帳號
   - 系統會自動發送邀請 Email

### 3.4 匯入庫存資料

**由診所管理員執行**：

1. **建立產品項目**
   - 前往「庫存管理」→「新增產品」
   - 填寫產品資訊：產品名稱、分類、單位、安全庫存

2. **匯入庫存數量**
   - 前往「庫存管理」→「匯入庫存」
   - 下載 Excel 範本
   - 填寫產品名稱與庫存數量
   - 上傳 Excel 檔案

## 多診所管理

### 6.1 多診所架構

**資料隔離機制**：
- 每個診所擁有獨立的資料空間
- 診所間資料完全隔離，無法互相存取
- 超級管理員可切換診所進行管理

**共用資源**：
- 系統設定（可選擇是否共用）
- 療程模板（可選擇是否共用）
- 報表模板（可選擇是否共用）

### 6.2 連鎖診所管理

**適用場景**：
- 同一品牌的多家分店
- 需要統一管理與數據彙總

**功能特色**：
- **總部管理介面**：統一管理所有分店
- **跨診所資料同步**：同步客戶資料、療程資料
- **多診所業績統計**：彙總所有分店業績
- **統一品牌設定**：所有分店使用相同品牌設定

**設定步驟**：
1. 建立總部組織
2. 建立分店並關聯至總部
3. 設定資料同步規則
4. 設定跨診所權限

### 6.3 獨立診所管理

**適用場景**：
- 不同品牌的診所
- 不需要資料共用

**功能特色**：
- **完全獨立**：每個診所完全獨立運作
- **獨立品牌**：每個診所使用自己的品牌設定
- **獨立帳號**：每個診所擁有獨立的管理員帳號

## 常見問題 FAQ

### Q1: 資料匯入失敗怎麼辦？

**A**: 請檢查以下項目：
1. 檔案格式是否正確（Excel 或 CSV）
2. 必填欄位是否已填寫
3. 資料格式是否符合範本（例如日期格式、電話格式）
4. 檔案大小是否超過限制（建議每批 500-1000 筆）

若仍無法解決，請下載錯誤報告並聯繫技術支援。

### Q2: LINE Bot 無法推播訊息？

**A**: 請檢查以下項目：
1. Webhook URL 是否正確設定
2. Channel Access Token 是否正確
3. LINE 訊息配額是否已用完
4. 用戶是否已加入好友

### Q3: 金流支付測試失敗？

**A**: 請檢查以下項目：
1. 是否使用測試模式
2. API Key 是否正確
3. 是否使用測試卡號（正式卡號無法在測試模式使用）
4. 網路連線是否正常

### Q4: 員工無法登入系統？

**A**: 請檢查以下項目：
1. Email 是否正確（登入帳號）
2. 是否已收到邀請 Email
3. 是否已設定密碼
4. 帳號是否已啟用

### Q5: 如何新增自訂角色？

**A**: 
1. 前往「角色權限管理」→「新增角色」
2. 填寫角色名稱
3. 勾選權限項目
4. 儲存角色
5. 在員工管理中指派新角色

### Q6: 如何匯出報表？

**A**: 
1. 前往對應的報表頁面
2. 設定篩選條件（時間範圍、診所等）
3. 點擊「匯出」按鈕
4. 選擇匯出格式（CSV、Excel、PDF）
5. 下載檔案

### Q7: 如何備份資料？

**A**: 
系統已設定每日自動備份，備份檔案保存 30 天。若需手動備份：
1. 前往「系統設定」→「資料備份」
2. 點擊「立即備份」按鈕
3. 等待備份完成
4. 下載備份檔案

### Q8: 如何還原資料？

**A**: 
1. 前往「系統設定」→「資料備份」
2. 選擇要還原的備份檔案
3. 點擊「還原」按鈕
4. 確認還原操作
5. 等待還原完成

**注意**：還原操作會覆蓋現有資料，請謹慎操作。

### Q9: 如何升級 LINE 訊息配額？

**A**: 
1. 前往 [LINE Official Account Manager](https://manager.line.biz/)
2. 選擇要升級的官方帳號
3. 前往「設定」→「方案」
4. 選擇適合的方案（Light、Standard、Pro）
5. 完成付款

### Q10: 如何聯繫技術支援？

**A**: 
1. 使用 Management UI 的 Feedback 功能
2. 或訪問 https://help.manus.im
3. 緊急問題可透過 Management UI 的 Support 功能聯繫技術團隊
